### 문제 이해
- 제재 아이디 중 일부 문자를 '*' 문자로 가려서 전달
- 문자 하나에 '*' 하나 이용하였고 아이디 당 최소 하나 이상의 '*' 문자 사용
- 당첨에서 제외되어야 할 제재 아이디 목록은 몇가지 경우의 수가 가능한지
#### input & output
- input:
    - 이벤트 응모자 아이디 목록이 담긴 배열 user_id
    - 불량 사용자 아이디 목록이 담긴 배열 banned_id
- output: 제재 아이디 목록의 경우의 수
