### 문제 이해
- 집 N개가 수직선 위에 있음
- 각각의 집의 좌표는 x1, ..., xN이고. 집 여러개가 같은 좌표를 가지는 일은 없음
- 집에 공유기 C개를 설치하려고 함
- 한 집에는 공유기를 하나만 설치할 수 있고, 가장 인접한 두 공유기 사이의 거리를 가능한 크게 하여 설치하려고 함
- C개의 공유기를 N개의 집에 적당히 설치해서, 가장 인접한 두 공유기 사이의 거리를 최대로 하는 프로그램
#### input & output
- input:
    - 집의 개수 N, 공유기 개수 C
    - 집의 좌표
- output: 가장 인접한 두 공유기 사이의 최대 거리
---
### 고안 방식
#### 최종 방식
- 집 사이의 거리를 이용
- 거리를 늘려가면서 총 C개 이상의 공유기를 설치할 수 있는지 확인
- 최대 거리 출력
