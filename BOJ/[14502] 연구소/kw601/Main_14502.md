# 백준 14502 연구소
## 입력
N M
N: 지도의 모양
0: 빈칸
1: 벽
2: 바이러스 위치( 2 <= n <= 10)

## 출력
안전 영역의 최대 크기

## 이해
바이러스: 상하좌우 인접한 칸을 모두 채움

벽: 칸 하나를 차지
3개의 벽을 세움

벽 3개 세운 후, 바이러스가 퍼질 수 없는 곳: 안전구역

안전구역이 가장 큰 경우 출력

## 생각
완탐으로 가능할까?
(8 * 8)C3 선택해서 (8*8) 탐색 -> (64 63 62/ 3 2 1)*64 *2(cnt)? 될 것 같기도

조합으로 가능한 경우 모두 하고 -> dfs돌려서 탐색, 한 번 쭉 돌면서 0 cnt
