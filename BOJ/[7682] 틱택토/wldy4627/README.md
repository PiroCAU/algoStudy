### 문제 이해
- 두 사람이 번갈아가며 X 또는 O 말을 놓는 게임
- 게임판은 3*3 격자판이며, 처음에는 비어 이씀
- 첫 번째 사람이 X를 놓고, 두 번째 사람이 O를 놓음
- 한 사람이 가로, 세로, 대각선 방향으로 3칸을 잇는 데 성공하거나 게임판이 가득 차면 게임은 끝남
- 게임판의 상태가 주어지면 그 상태가 틱택토 게임에서 발생할 수 있는 최종 상태인지 판별
#### input & output
- input: 여러 개의 테스트 케이스, 마지막을 나타내는 문자열 "end"
- output: 각 테스트 케이스마다 가능하다면 valid, 불가능하다면 invalid

#### 최종 방식
- X의 개수는 O의 개수와 같거나 1개 더 많아야 함 -> valid
- X의 개수가 O보다 1개 더 많고, O의 빙고가 없음 -> X가 이김
- X와 O의 개수가 같고, X의 빙고가 없음 -> O가 이김
- 보드가 꽉 차거나 빙고가 없음 -> 무승부
