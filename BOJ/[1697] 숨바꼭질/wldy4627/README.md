### 문제 이해
- 수빈이는 N에 있고, 동생은 K에 있음
- 수빈이는 걷거나 순간이동을 할 수 있음
  - 수빈이의 위치가 X일 때 걷는다면 1초 후에 X-1 또는 X+1로 이동함
  - 순간이동을 하는 경우에는 1초 후에 2*X로 이동함
- 수빈이와 동생의 위치가 주어졌을 때, 수빈이가 동생을 찾을 수 있는 가장 빠른 시간이 몇 초 후인지 구하는 프로그램
#### input & output
- input: N, K
- output: 수빈이가 동생을 찾는 가장 빠른 시간
---
### 고안 방식
- BFS
#### 최종 방식
- 3가지 방식 모두 진행
