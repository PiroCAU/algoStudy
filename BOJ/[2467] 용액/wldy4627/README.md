### 문제 이해
- 각 용액에는 그 용액의 특성을 나타내는 하나의 정수가 주어져있음
- 산성 용액은 1부터 양의 정수로, 알칼리성 용액은 -1부터 음의 정수로 나타냄
- 두 용액을 혼합한 용액의 특성값은 혼합에 사용된 각 용액의 특성값의 합으로 나타냄
- 같은 양의 두 용액을 혼합해 특성값이 0에 가장 가까운 용액을 만들고자 함
- 산성 용액과 알칼리성 용액의 특성값이 정렬된 순서로 주어졌을 때, 두 개의 서로 다른 용액을 혼합해 특성값이 0에 가장 가까운 용액을 만들어내는 두 용액을 찾는 프로그램
#### input & output
- input: 전체 용액의 수 N, 용액의 특성값을 나타내는 N개의 정수
- output: 0에 가장 가까운 용액을 만들어내는 두 용액의 특성값(오름차순)
