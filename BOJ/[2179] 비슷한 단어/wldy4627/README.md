### 문제 이해
- N개의 영단어에서 가장 비슷한 두 단어를 구해내는 프로그램
- 비슷한 정도: 두 단어의 접두사의 길이
- 두 단어의 앞에서부터 M개의 글자들이 같으면서 M이 최대인 경우
- 접두사의 길이가 최대인 경우가 여러 개일 때에는 입력되는 순서대로 제일 앞쪽에 있는 단어가 답
  - 그런 경우도 여러 개 있을 경우 T가 입력되는 순서대로 제일 앞쪽에 있는 단어인 경우
#### input & output
- input:
    - N
    - 알파벳 소문자로 이루어진 길이 100자 이하의 서로 다른 영단어
- output: S, T (서로 달라야 함)
---

#### 최종 방식
- 정렬 -> 뒤에 것과 비교
- M이 최대인 것을 구하고, 해당 Prefix를 가지는 단어들 중 적절한 S, T 찾기
