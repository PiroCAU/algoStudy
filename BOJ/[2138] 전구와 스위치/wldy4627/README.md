### 문제 이해
- N개의 스위치, N개의 전구
- 각 전구는 꺼져 있거나 켜져 있는 상태를 가짐
- i번 스위치를 누르면 i-1, i, i+1의 세 개의 전구의 상태가 바뀜
    - 꺼져 있는 전구는 켜지고, 켜져 있는 전구는 꺼짐
- N개의 전구들의 현재 상태와 우리가 만들고자 하는 상태가 주어졌을 때, 그 상태를 만들기 위해 스위치를 최소 몇 번 누르면 되는지 알아내는 프로그램
#### input & output
- input
    - N
    - 전구들의 현재 상태 (0: 켜짐, 1: 꺼짐)
    - 만들고자 하는 전구들의 상태
- output: 원하는 상태를 만들기 위해 눌러야 하는 스위치의 최소 횟수 (불가능할 경우 -1)

- 000 -> 110 -> 001 -> 010

- 이전 자릿수가 맞으면 스위치 누르면 안됨
