### 문제 이해
- 세로 두 줄, 가로로 N개의 칸으로 이루어진 표
- 첫째 줄의 각 칸에는 정수 1, 2, ..., N이 차례대로 들어 있음
- 둘째 줄의 각 칸에는 1이상 N이하인 정수가 들어 있음
- 첫째 줄에서 숫자를 적절히 뽑으면, 그 뽑힌 정수들이 이루는 집합과, 뽑힌 정수들의 바로 밑의 둘째 줄에 들어있는 정수들이 이루는 집합이 일치함
- 이 조건을 만족시키도록 정수들을 뽑되, 최대로 많이 뽑는 방법을 찾는 프로그램
#### input & output
- input: N, 둘째 줄에 들어가는 정수
- output: 뽑힌 정수들의 개수, 뽑힌 정수들을 오름차순으로
---
### 알고리즘 분류
- dfs, 그래프
#### 최종 방식
- 인덱스와 두번째 줄 숫자를 연결했을 때, 시작점으로 다시 돌아오는 사이클 찾기
- 모든 숫자를 시작점으로 해서 DFS 수행 -> 사이클이 확인되면 저장 -> 가장 많은 것을 출력
