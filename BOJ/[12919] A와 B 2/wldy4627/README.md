### 문제 이해
- 문자열 S와 T가 주어졌을 때, S를 T를 바꾸고자 함. 바꿀 때는 다음과 같은 두 가지 연산만 가능함
  1. 문자열의 뒤에 A를 추가함
  2. 문자열의 뒤에 B를 추가하고 문자열을 뒤집음
- 주어진 조건을 이용해서 S를 T로 만들 수 있는지 알아내기
#### input & output
- input: S, T
- output: S를 T로 바꿀 수 있으면 1을, 없으면 0 출력
---
### 알고리즘 분류
- 브루트포스, 재귀
#### 최종 방식
- 길이가 같을 때, 두 문자열이 동일하면 1, 다르면 0
- T의 첫 글자가 B라면 -> B를 떼고 뒤집기
- T의 마지막 글자가 A라면 -> A 떼기
