### 문제 이해
- N명의 사람들은 매일 아침 한 줄로 섬
- 사람들은 자기보다 큰 사람이 왼쪽에 몇 명 있었는지만 기억함
- N명의 사람이 있고, 사람들의 키는 1부터 N까지 다름
- 각 사람들이 기억하는 정보가 주어질 때, 줄을 어떻게 서야하는지 출력하는 프로그램
#### input & output
- input
    - N
    - 키가 1인 사람부터 자기보다 키가 큰 사람이 왼쪽에 몇 명이 있었는지 주어짐
- output: 줄을 선 순서대로 키를 출력
