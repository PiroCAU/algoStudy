### 문제 이해
- 대회에서 총 k개의 문제를 풂
- 어떤 문제에 대해 풀이를 제출하면 0~100점의 점수를 얻음
- 풀이를 제출한 팀의 ID, 문제 번호, 점수가 서버의 로그에 제출되는 시간 순서대로 저장됨
- 한 문제에 대한 풀이를 여러 번 제출할 수 있고, 최고 점수가 그 문제의 최종 점수가 됨
- 팀의 최종 점수는 각 문제에 대해 받은 점수의 총합이고, 순위는 (팀보다 높은 점수를 받은 팀의 수) + 1임
- 점수가 동일한 팀이 여럿이라면, 다음 규칙에 의해 순위가 정해짐
  1. 최종 점수가 같은 경우, 풀이 제출 횟수가 적은 팀의 순위가 높음
  2. 최종 점수도 같고 제출 횟수도 같은 경우, 마지막 제출 시간이 더 빠른 팀의 순위가 높음
- 동시에 제출되는 풀이는 없고, 모든 팀이 적어도 한번은 풀이를 제출
- 서버 로그가 주어졌을 때, 당신 팀의 순위를 계산하는 프로그램
#### input & output
- input: T개의 테스트 데이터로 구성됨
    - 테스트 데이터 수 T
    - 팀의 개수 n, 문제의 개수 k, 당신 팀의 ID t, 로그 엔트리의 개수 m
    - 각 풀이에 대한 정보 -> 팀 ID i, 문제 번호 j, 획득 점수 s
- output: 고이는 빗물의 총량
