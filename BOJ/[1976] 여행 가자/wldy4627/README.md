### 문제 이해
- 한국에는 도시가 N개 있음
- 임의의 두 도시 사이에 길이 있을 수도, 없을 수도 있음
- 중간에 다른 도시를 경유해서 여행을 할 수도 있음
- 여행 일정이 주어졌을 때, 이 여행 경로가 가능한지 판별하는 프로그램
- 같은 도시를 여러 번 방문하는 것도 가능함
#### input & output
- input:
    - 도시의 수 N
    - 여행 계획에 속한 도시들의 수 M
    - N개의 정수: i번째 줄의 j번째 수 = i번 도시와 j번 도시의 연결 정보
      - 1이면 연결 / 0이면 연결 안 됨
    - 여행 계획
- output: 가능하면 YES, 불가능하면 NO
