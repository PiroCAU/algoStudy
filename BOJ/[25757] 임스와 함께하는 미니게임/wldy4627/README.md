### 문제 이해
- 미니게임: 윷놀이 Y(2), 같은 그림 찾기 F(3), 원카드 O(4)
- 인원 수가 부족하면 게임을 시작할 수 없음
- 임스와 같이 플레이하기를 신청한 횟수 N과 임스가 플레이할 게임의 종류가 주어질 때, 최대 몇 번이나 임스와 함께 게임을 플레이할 수 있는지
- 임스는 한 번 같이 플레이한 사람과는 다시 플레이하지 않음
- 임스와 함께 플레이하고자 하는 사람 중 동명이인은 존재하지 않음
#### input & output
- input: 같이 플레이하기를 신청한 횟수 N, 같이 플레이할 게임의 종류, 같이 플레이하고자 하는 사람들의 이름
- output: 최대로 몇 번이나 게임을 플레이할 수 있는지
