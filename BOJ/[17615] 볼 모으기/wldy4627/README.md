### 문제 이해
- 볼을 옮겨서 같은 색 볼끼리 인접하게 놓이도록 하려고 함
- 볼을 옮기는 규칙은 다음과 같음
  1. 바로 옆에 다른 색깔의 볼이 있으면 그 볼을 모두 뛰어 넘어 옮길 수 있음
  2. 옮길 수 있는 볼의 색깔은 한 가지임
- 일직선상에 놓여 있는 볼에 관한 정보가 주어질 때, 규칙에 따라 볼을 이동해 같은 색끼리 모으되 최소 이동횟수를 찾는 프로그램
#### input & output
- input
    - 볼의 총 개수 N
    - 볼의 색깔을 나타내는 문자 R(빨간색 볼) 또는 B(파란색 볼)
- output: 최소 이동 횟수
---

- 그리디
#### 최종 방식
- R을 오른쪽으로 몰기
- R을 왼쪽으로 몰기
- B를 오른쪽으로 몰기
- B를 왼쪽으로 몰기
