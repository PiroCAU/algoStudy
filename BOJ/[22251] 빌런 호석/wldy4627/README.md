### 문제 이해
- 1층부터 N층까지 이용이 가능한 엘리베이터
- 엘리베이터 층수를 보여주는 디스플레이에는 K자리의 수가 보임
- 수는 0으로 시작할 수 있음
- 각 숫자는 7개의 표시등 중의 일부에 불이 들어오면서 표현됨
- 빌런 호석은 치르보기 빌딩의 엘리베이터 디스플레이의 LED 중에서 최소 1개, 최대 P개를 반전시키고자 함 
- 반전이란 켜진 부분은 끄고, 꺼진 부분은 켜는 것을 의미함 
- 반전 이후에 디스플레이에 올바른 수가 보여지면서 1 이상 N 이하가 되도록 바꾸고자 함 
- 현재 엘리베이터가 실제로 X층에 멈춰있을 때, 반전시킬 LED를 고를 수 있는 경우의 수
#### input & output
- input: N, K, P, X
- output: 호석 빌런이 LED를 올바르게 반전시킬 수 있는 경우의 수
