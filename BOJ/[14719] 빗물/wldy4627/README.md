### 문제 이해
- 2차원 세계에 블록이 쌓여있음
- 비가 오면 블록 사이에 빗물이 고임
- 비는 충분히 많이 올 때, 고이는 빗물의 총량은?
#### input & output
- input:
    - 2차원 세계의 세로 길이 H, 가로 길이 W
    - 블록이 쌓인 높이가 2차원 세계의 맨 왼쪽 위치부터 차례대로 W개 주어짐
- output: 고이는 빗물의 총량
---
### 알고리즘 분류
- 구현
#### 최종 방식
- 물이 고이기 위해서는 왼쪽과 오른쪽에 자신보다 높은 벽이 있어야함
- 자기 위치 기준 min(왼쪽에서의 최대 높이, 오른쪽에서의 최대 높이) - 자기 위치 만큼 물이 쌓일 수 있음
- 음수면 0
