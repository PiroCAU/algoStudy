### 문제 이해
- N장의 카드. 각 카드는 1부터 N까지의 번호가 붙어 있음
- 1번 카드가 제일 위, N번 카드가 제일 아래인 상태로 순서대로 카드가 놓여 있음
- 카드가 한 장 남을 때까지 다음 동작을 반복
  1. 제일 위에 있는 카드를 바닥에 버림
  2. 제일 위에 있는 카드를 제일 아래에 있는 카드 밑으로 옮김
- N이 주어졌을 때, 제일 마지막에 남게 되는 카드를 구하는 프로그램
#### input & output
- input: 정수 N
- output: 남게 되는 카드의 번호
---
### 알고리즘 분류
- Queue 이용
#### 최종 방식
- 선입선출 Queue를 이용
- Queue에서 하나 빼고, 두번째꺼는 빼서 뒤에 다시 삽입
