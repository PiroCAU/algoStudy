### 문제 이해
- 회전하는 벨트 위에 여러 종류의 초밥이 접시에 담겨 놓여 있고, 손님은 이 중에서 자기가 좋아하는 초밥을 골라서 먹음
- 벨트 위에는 같은 종류의 초밥이 둘 이상 있을 수 있음
- 두가지 행사를 진행함
  1. 벨트의 임의의 한 위치부터 k개의 접시를 연속해서 먹을 경우 할인된 정액 가격으로 제공 
  2. 각 고객에게 초밥의 종류 하나가 쓰인 쿠폰을 발행하고, 1번 행사에 참가할 경우 이 쿠폰에 적혀진 종류의 초밥 하나를 추가로 무료로 제공. 만약 이 번호에 적혀진 초밥이 현재 벨트 위에 없을 경우, 요리사가 새로 만들어 손님에게 제공.
- 가능한 한 다양한 종류의 초밥을 먹으려고 함. 
- 벨트 상태, 초밥의 가짓수, 연속해서 먹는 접시의 개수, 쿠폰 번호가 주어졌을 때, 손님이 먹을 수 있는 초밥 가짓수의 최댓값을 구하는 프로그램
#### input & output
- input
    - 회전 초밥 벨트에 놓인 접시의 수 N, 초밥의 가짓수 d, 연속해서 먹는 접시의 수 k, 쿠폰 번호 c
    - 초밥의 종류
- output: 회전 초밥 벨트에서 먹을 수 있는 초밥의 가짓수의 최댓값
