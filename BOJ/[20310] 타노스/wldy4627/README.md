### 문제 이해
- 0과 1로 이루어진 문자열 S가 있음
- S가 포함하는 0의 개수와 1의 개수는 모두 짝수임
- S를 구성하는 문자 중 절반의 0과 절반의 1을 제거해 새로운 문자열 S'를 만들고자 함
- S'로 가능한 문자열 중 사전순으로 가장 빠른 것
- 서브 태스크: S의 길이는 4의 배수, S의 홀수 번째 문자는 1, 짝수 번째 문자는 0
#### input & output
- input: 문자열 S
- output: S'로 가능한 문자열 중 사전순으로 가장 빠른 것

#### 최종 방식
- 사전순으로 가장 빠르기 위해서는 가능하다면 최대한 0이 맨 앞으로 가도록
- 최대한 앞에서 1의 절반 개수를 제거
- 최대한 뒤에서 0의 절반 개수를 제거
- 111100 -> 110
- 110101 -> 011
- 101111000001 -> 011001
- 111110000001 -> 110001
