### 문제 이해
- 1부터 N까지의 수를 오름차순으로 쓴 수열 1 2 3 ... N
- '+'나 '-', 또는 ' '(공백)을 숫자 사이에 삽입(+는 더하기, -는 빼기, 공백은 숫자를 이어 붙이는 것을 뜻함)
- 수식의 값을 계산하고 그 결과가 0이 될 수 있는지를 확인해보자 
- N이 주어졌을 때 수식의 결과가 0이 되는 모든 수식을 찾는 프로그램
#### input & output
- input:
  - 테스트 케이스 개수
  - 자연수 N
- output: ASCII 순서에 따라 결과가 0이 되는 모든 수식. 각 테스트 케이스의 결과는 한 줄을 띄워 구분
