### 이해

화살 n발씩 쏨
k점을 어피치가 a발, 라이언이 b발
-> a >= b면 어피치, 아니면 라이언이 k점
if a == b == 0 -> 0점

모든 과녁(1~10)에 대해 점수 계산

어피치 >= 라이언이면 어피치 승

라이언은 어피치를 가장 큰 점수 차로 이기기 위해 n발을 어디에 맞혀야 하는지 구함
우승할 수 없으면 [-1]
10점부터 순서대로 점수를 리스트에 담아 return
