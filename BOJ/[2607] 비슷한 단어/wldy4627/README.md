### 문제 이해
- 영문 알파벳 대문자로 이루어진 두 단어가 다음 조건을 만족하면 같은 구성을 가짐
  1. 두 단어가 같은 종류의 문자로 이루어짐
  2. 같은 문자는 같은 개수만큼 있음
- 하나의 문자를 다른 문자로 바꾸거나, 더하거나, 빼서 다른 한 단어와 같은 구성을 갖게 되는 경우 두 단어를 서로 비슷한 단어라고 함
- 입력으로 여러 개의 단어가 주어질 때, 첫번째 단어와 비슷한 단어가 몇 개인지 출력하는 프로그램
#### input & output
- input: 단어의 개수, 단어
- output: 첫 번째 단어와 비슷한 단어가 몇 개인지
