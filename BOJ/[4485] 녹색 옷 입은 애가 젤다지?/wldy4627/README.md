### 문제 이해
- 화폐의 단위는 루피(rupee)
- '도둑루피'라 불리는 검정색 루피를 획득하면 소지한 루피가 감소
- 링크는 도둑루피만 가득한 N x N 크기의 동굴의 제일 왼쪽 위에 있음([0][0]번 칸)
- 링크는 이 동굴의 반대편 출구, 제일 오른쪽 아래 칸인 [N-1][N-1]까지 이동해야 함
- 동굴의 각 칸마다 도둑루피가 있는데, 이 칸을 지나면 해당 도둑루피의 크기만큼 소지금을 잃게 됨
- 링크는 잃는 금액을 최소로 하여 동굴 건너편까지 이동해야 하며, 한 번에 상하좌우 인접한 곳으로 1칸씩 이동할 수 있음 
- 링크가 잃을 수밖에 없는 최소 금액은 얼마일까?
#### input & output
- input
    - 동굴의 크기를 나타내는 정수 N (0일 경우 종료)
    - 동굴의 각 칸에 있는 도둑루피의 크기
- output: 링크가 잃는 최소 금액
