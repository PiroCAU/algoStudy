### 문제 이해
- 한 줄로 된 간단한 에디터
- 영어 소문자만을 기록할 수 있는 편집기로, 최대 600,000글자까지 입력 가능
- 커서는 문장의 맨 앞, 문장의 맨 뒤, 문장 중간 임의의 곳에 위치할 수 있음
- 길이가 L인 문자열이 현재 편집기에 입력되어 있으면, 커서가 위치할 수 있는 곳은 L+1가지 경우가 있음
- 편집기가 지원하는 명령어
  1. L: 커서를 왼쪽으로 한 칸 (커서가 문장의 맨 앞이면 무시)
  2. D: 커서를 오른쪽으로 한 칸 (커서가 문장의 맨 뒤이면 무시)
  3. B: 커서 왼쪽에 있는 문자 삭제 (커서가 문장의 맨 앞이면 무시)
  4. P $: $라는 문자를 커서 왼쪽에 추가
- 모든 명령어를 수행하고 난 후 편집기에 입력되어 있는 문자열을 구하는 프로그램
- 명령어 수행되기 전에 커서는 문장의 맨 뒤에 있음
#### input & output
- input
    - 초기 편집기에 입력되어 있는 문자열 (길이 N)
    - 입력할 명령어의 개수 M
    - 입력할 명령어
- output: 편집기에 입력되어 있는 문자열
